# WezTerm Keybindings Reference

> **Hyper Key** = `Ctrl+Alt+Shift` (mapped to CapsLock)

---

## Tabs

| Action | Keybind |
|--------|---------|
| New tab | `Hyper + t` |
| Close tab | `Hyper + w` |
| Next tab | `Hyper + n` or `Hyper + Tab` |
| Previous tab | `Hyper + p` |
| Jump to tab 1 | `Hyper + 1` |
| Jump to tab 2 | `Hyper + 2` |
| Jump to tab 3 | `Hyper + 3` |
| Jump to tab 4 | `Hyper + 4` |
| Jump to tab 5 | `Hyper + 5` |
| Jump to tab 6 | `Hyper + 6` |
| Jump to tab 7 | `Hyper + 7` |
| Jump to tab 8 | `Hyper + 8` |
| Jump to tab 9 | `Hyper + 9` |
| Move tab left | `Hyper + {` |
| Move tab right | `Hyper + }` |
| Rename tab | `Hyper + r` |

---

## Panes

### Creating & Closing

| Action | Keybind |
|--------|---------|
| Split horizontal | `Hyper + d` |
| Split vertical | `Hyper + s` |
| Close pane | `Hyper + x` |
| Zoom/unzoom pane | `Hyper + z` |

### Navigation (Vim-style)

| Action | Keybind |
|--------|---------|
| Move left | `Hyper + h` |
| Move down | `Hyper + j` |
| Move up | `Hyper + k` |
| Move right | `Hyper + l` |

### Resizing

| Action | Keybind |
|--------|---------|
| Resize left | `Hyper + LeftArrow` |
| Resize right | `Hyper + RightArrow` |
| Resize up | `Hyper + UpArrow` |
| Resize down | `Hyper + DownArrow` |

---

## Search & Selection

| Action | Keybind |
|--------|---------|
| Search scrollback | `Hyper + f` |
| Copy mode (vim-like) | `Hyper + v` |
| Quick select (URLs, paths) | `Hyper + q` |

---

## Utilities

| Action | Keybind |
|--------|---------|
| Command palette | `Hyper + Space` |
| Debug overlay | `Hyper + F12` |
| Clear scrollback | `Hyper + Backspace` |

---

## Simple Keybinds (Frequent Actions)

These use simpler modifiers for speed:

### Copy & Paste

| Action | Keybind |
|--------|---------|
| Copy | `Ctrl + Shift + c` |
| Paste | `Ctrl + Shift + v` |

### Font Size

| Action | Keybind |
|--------|---------|
| Increase font | `Ctrl + =` |
| Decrease font | `Ctrl + -` |
| Reset font | `Ctrl + 0` |

### Scrolling

| Action | Keybind |
|--------|---------|
| Page up | `Shift + PageUp` |
| Page down | `Shift + PageDown` |

---

## Config Info

- **Shell**: PowerShell (`pwsh.exe`)
- **Theme**: Tokyo Night (Gogh)
- **Font**: 0xProto Nerd Font @ 14pt
- **Transparency**: 85% with Acrylic blur
- **Scrollback**: 5000 lines
- **Inactive panes**: Dimmed (24% saturation, 50% brightness)
